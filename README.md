# User-Authentication-System

##  Description
This is a Java-based user authentication system that allows users to register and log in securely using hashed passwords.

##  Features
- User Registration
- User Login
- Password Hashing for security
- Simple and clean console interface

##  Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Hashing (for password security)

##  How to Run
1. Open the project in IntelliJ IDEA
2. Run `LoginSystem.java`
3. Follow the console instructions

- How authentication systems work
- Password hashing and security basics
- Java OOP concepts
